1. 无理由退换货

    “无理由退换货”是指您在购买支持无理由退换货的商品后，自实际收货日期的次日起7天内，商品完好的情况下，可无理由退换货（部分商品及类目除外，详见各品类细则）需由您承担商品发出与返回的运费。

    1、未收货或收货前想退货,商品完好，您需要承担已产生的运费；

    2、收货验货时对商品不满意，开箱验货外观完好，可以选择退换货，您需要承担已产生的运费；收货验货时外观破损瑕疵、缺漏附件等问题，您可指出并选择退换货，不需要承担运费。

    3、以下商品不享受“7天无理由退货”政策： 1）签收后对商品不满意未拆包装，自收货次日起7天（不支持无理由换同型） 2）已拆包装未使用或仅试用（无人为损坏），自收货次日起7天（不支持无理由换同型） 3）已安装使用存在性能故障（无人为损坏），自收货次日起确认故障存在，7天退15天换 4）已拆包装商品有人为损坏，不支持退换 5）无需安装产品性能或质量问题退换货，自收货次日起确认故障存在，7天退15天换

  2. 备注：

    1.不支持收货后办理退换货的品类除外

    2.无质量问题或因您自身原因造成的退货，由您承担退货产生运费

    3.已安装的商品，出现性能故障，在故障经过确认后即可免费办理退换货

    4.因商品使用后的污损、磕碰、划伤等影响商品完好的情形，则不支持无理由退换货

    5.您可在订单中心自助申请退/换货，因质量问题产生的退、换货订购人需选择快递方式将商品寄回，并先行垫付快递费用，超级积分确认产品有质量问题后,再将运费返至订购人帐户中,超级积分不接收受以到付形式寄回的包裹

  3. 配送与签收：

    (1)在您收到商品时，如无需安装，请务必检查是否存在因运输过程造成的损坏、漏液、破碎等外观损坏，或原装配件缺失、缺少发票等缺少附件问题，如您发现以上情况，请选择拒收，并可第一时间与超级积分或苏宁客服联系；需安装类商品请在安装工人上门后开箱验机，货物签收后安装工人未上门前请勿拆箱。部分城市会有所差异，请按照物流工人指导进行验机、收货

    (2)如因您个人或您授权的其他个人代签收的商品，视同您已在收货时确认商品完好

  4. 退换货商品附件要求：

    退换货(包含有质量问题的商品）时，请务必将商品的内带附件、赠品（如有）、保修卡、说明书、发票、检测报告（针对需凭检测报告办理退换货的商品）等随同商品一起退回

    友情提示：建议产品外包装自收货之日起保留15日，退换货时附商品的原外包装

  5. 以下情况，不予办理退换货：

    (1)任何非超级积分出售的商品（序列号不符）

    (2)过保商品（超过退换货期限的商品）

    (3)未经授权的维修、误用、碰撞、疏忽、滥用、进液、事故、改动、不正确的安装所造成的商品质量问题，或撕毁、涂改标贴、机器序号、防伪标记

    (4)商品的附件、赠品（需要和主商品一起退换）、发票缺失或涂改等影响商品完好的情形

    (5)购买时明确说明不支持退货的商品

    (6)其他依法不应办理退换货的

  6. 以下商品因特殊性，不参加无理由退换货：

    (1)鲜活易腐的商品

    (2)在线下载或者您拆封的音像制品、计算机软件等数字化商品

    (3)服装鞋帽类商品已洗涤的

    (4)实物类商品：通讯合约机、艺术品及收藏品、奢侈品、美妆/个人洗护类商品、食品/饮料/酒水、奶粉/辅食/尿布/、望远镜/显微镜/夜视镜、隐形眼镜、贴身使用的成人婴儿衣物、钻石/黄金/手表/珠宝首饰及个人配饰类产品

    (5)无法保证退回商品完好的

    (6）其他根据商品性质不宜适用无理由退换货的商品

  7. 各种产品类别退、换货细则

    服饰、箱包及鞋类商品退货细则 1. 出于安全和卫生考虑，贴身用品如内衣裤、袜子、帽子等商品，除质量问题外，概不退货。 2. 任何因非正常使用、保管或个人原因造成的商品损坏（如自行修改尺寸，洗涤，皮具表面刮花、打油，刺绣，碰酸、碱、油或者触硬物，雨天穿着，五金配件缺少等），不予退货。 3. 商品吊牌、发货单、商品配件（如配饰挂坠、肩带、小卡包等）、说明书、保修单、标签等丢失，不予退货。 4. 商品及商品本身包装保持出售时原状且配件齐全，未撕下吊牌，不影响二次销售，在7天无理由退货期限内可以办理退货。 5. 遇过敏问题申请退货需要提供医院的相关证明。

    美容化妆类商品退货细则 1. 出于安全和卫生考虑，贴身用品、化妆品不予退货，经权威部门检测商品存在质量问题者除外。 2. 如签收时商品包装破损，商品已过期或离过期不到2个月，可以申请退货。 3. 遇过敏问题申请退货需要提供医院的相关证明。 4. 温馨提示：化妆品的使用效果会因肤质、年龄、季节、气候、使用方法以及配套产品等的不同而效果不一，因此，介绍的功能及使用效果仅供参考，不能作为退货依据。

    家居家纺类商品退货细则 1. 因个人原因造成商品损坏（如自行修改尺寸、洗涤）的，不予退货。。 2. 商品吊牌、发货单、商品配件、说明书、保修单及标签等丢失的或包装破损，不予退货。

    小家电类商品退货细则 1. 商品及商品本身包装保持出售时原状且配件赠品资料齐全(注： 未打开塑封包装和商品外包装不影响二次销售，可在签收后7个自然日内退货。） 2. 退货时，请务必将商品的外包装、内带附件、保修卡、说明书、等随同商品一起退回。 3. 若商品在购买时附带赠品，办理退货时需将赠品一起退回。 4. 因厂商可能会在未提前通知的情况下更改商品包装、产地或者一些附件，因此收到的商品有可能与超级积分提供的图片、产地、附件及附件说明不完全一致，请您以收到商品实物中标示的包装清单为准，超级积分会尽量做到及时更新商品包装、产地和附件说明的介绍。 5. 个人护理相关商品，如：剃须刀、女士剃/脱毛器、美容器、卷发器、理发器、电动牙刷、鼻毛修剪器等，请当场验收，若商品有破损、渗漏、缺少、配送错误等问题，请及时与超级积分客服联系。

  8. 退、换货方式：

    1. 登陆超级积分网站、找到订单，在网站上按要求申请退、换货。客服人员会在收到申请后的一个工作日内联系您。

    2. 打开超级积分网站，点击在线客服，通过在线客服为您提交退、换货申请。
